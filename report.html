Report: Everything Milan

Bilyana Georgieva

Introduction

The site created is a great starting point for anyone wanting to get to know Milan a little bit better. I decided to focus on mostly cheaper places and locations with an exciting history that may not be so popular. All the articles I have written myself based on my personal experience and advice that locals have given me. The idea is to create a very simple webste with just the essential information so that the people can quickly scroll and find their next destination, seeing the opening hours and the pricing easily. Rather than bombarding them with information, we are letting them discover the city on their own with a little bit of guidance.

For the purpose of building the website, I used Handlebars.js for the HTML templates, jQuery for easier manipulation of the DOM and Bootstrap with Material theme for easier CSS.

Inspiration

  1. My personal experience in Milan
  2. The website Flawless Milano
  3. The bootstrap blog template

What I learned?

  1. Writing all the CSS by hand is a waste of time. Bootstrap and similar libraries are very useful and save tons of time. I was referred to Bootstrap by my mom who is a Javascript developer.
  2. My mom also referred me to jQuery which I personally find much easier to write and read in my code than vanilla JS.
  3. I also learned how to read parameters from the url from an article in StackOverflow which was very useful because I used in pretty much all of my pages. There I pretty much copied the code I found in a function and then called that function whenever I needed it.

What worked well?

  1. I was very efficient with my searching in StackOverflow. There wasn't ever a long period of time where I would search for something without an answer.
  2. I managed to easily find templates online for my html and tweak them according to my preference so that they fit my original idea.
  3. Once having finished the first page I managed to use elements from there in all the other pages so there was mostly tweaking the HTML and adding JS, rather than writing everything anew every time.
  4. Thanks to Bootstrap and the Material theme I didn't have to write almost any CSS!

What could I improve?

  1. I could add more animations.
  2. Currently my images are distorted at some places, I could've chosen a different approach of work and made them in the crrect ratio.
  3. I could've added more articles to showcase better the website's features.
